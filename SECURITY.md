# Política de seguridad

La rama `main` es la versión soportada.

Los avisos de seguridad deben enviarse de forma privada al propietario `@JoseLH23`. No publiques información confidencial en issues.

Indica el componente, la versión afectada y una explicación breve. Las comprobaciones sobre ambientes reales requieren autorización previa y no deben usar datos reales de visitantes.
